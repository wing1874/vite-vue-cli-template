# vue3+elementUI 脚手架

### 使用

# create-vue3-elementui

> Vite requires [Node.js](https://nodejs.org/en/) version 14.18+, 16+.

With NPM:

```bash
npm create vue3-elementui@latest
```

With Yarn:

```bash
yarn create vue3-elementui
```

With PNPM:

```bash
pnpm create vue3-elementui
```

Then follow the prompts
